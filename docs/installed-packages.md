# Installed Packages

## Backend

```bash
# class-transformer
# - JavaScript Plain Object と Class Object(instance) を相互に変換
# - JSON と JavaScript Plain Object を相互に変換
# class-validator
npm i class-validator class-transformer
# Mysql
npm install @nestjs/typeorm typeorm mysql2
# like dotenv package
npm i --save @nestjs/config
# set env(https://github.com/kentcdodds/cross-env)
npm i cross-env
# HTTP request
npm i got
# for form data
npm i form-data
# for cron
npm i --save @nestjs/schedule
# for image upload
npm i aws-sdk
```

## Frontend

```bash
# Already Installed when create project
# https://tailwindcss.com/docs/installation
npm install tailwindcss@latest postcss@latest autoprefixer@latest
# form
npm install react-hook-form
# font awesome
npm install --save @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons @fortawesome/react-fontawesome @fortawesome/free-brands-svg-icons @fortawesome/free-regular-svg-icons

# postpone -------------------------------------------------------
# document 의 head 를 변경할 수 있게 해준다.
npm install react-helmet-async @types/react-helmet-async
# cypress
npm install --save-dev cypress @testing-library/cypress
# chart
npm install --save victory-native
# google map
npm install google-map-react @types/google-map-react @types/googlemaps
```
